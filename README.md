# Grid Master

Десктопное приложение для производственных менеджеров на заводе металлических решётчатых настилов. Получаешь PDF-чертежи от заказчика — программа распознаёт позиции через Claude Vision, считает стоимость и формирует коммерческое предложение.

Дополнительный модуль **Steps Master** — расчёт ступеней из решётчатого настила.

---

## Возможности

- **Автораспознавание чертежей** — Claude Vision извлекает позиции, габариты, артикулы, покрытие, радиусные вырезы
- **Анализ обрамления по кромкам** — ИИ определяет тип обрамления на каждой стороне: стандартный уголок или кикплейт, считает длины и объясняет свой вывод
- **Сверка со спецификацией** — загружаешь Word/Excel/PDF от заказчика, программа сравнивает с распознанными позициями и помечает расхождения
- **Автоматический расчёт трудозатрат** — формула по периметру, дугам и косым резам с коэффициентом сложности
- **Полная стоимость** — мат, обрамление, перфоуголок, кикплейт, работа, цинкование
- **Раскрой матов** — упаковка деталей по листам с визуализацией
- **Экспорт** — PDF-оферта (EN/ET) и Excel для склада с двумя листами
- **Steps Master** — расчёт ступеней: боковины, решётка, перфоуголок, кикплейт, работа, цинк

## Первый запуск

При первом запуске программа попросит указать Excel-файлы с данными:

| Файл | Обязательные колонки |
|---|---|
| Маты | `Артикул`, `Цена м²`, `Вес м²`, `Длина мата` |
| Обрамление | `Размер`, `Вес` |
| Перфоуголок | `Размер`, `Вес` |
| Отбойник/Кикплейт | `Размер`, `Вес` |
| Боковины (ступени) | `Размер`, `Ширина`, `Вес`, `Цена` |

API-ключ Anthropic вводится в ⚙ Настройки. Всё сохраняется в `manager_config.json`.

## Рабочий процесс

```
1. История → + Новый заказ
2. Загрузить PDF чертежи → Распознать (Claude Vision)
3. Проверить позиции, при необходимости отредактировать
   - ИИ показывает анализ обрамления по кромкам (🤖)
   - Можно задать длины обрамления/кикплейта вручную
4. Нажать Рассчитать
5. Экспортировать: PDF оферта / Excel для склада
```

| `manager_app.py` | Весь UI. 4 экрана: история → загрузка → проверка → результат |

## Конфигурация

`manager_config.json` создаётся автоматически. Хранит:
- Пути к Excel-файлам и их mtime для автоперезагрузки при изменении
- Данные матов, весовые таблицы материалов
- Цены с памятью по типу (`type_prices`) — выбрал тип, цена подставилась
- Нормативы трудозатрат (`labour_norms`)
- Цены для ступенек (`steps_prices`)
- Последние выбранные типы материалов

---

## Нормативы трудозатрат

Хранятся в конфиге, редактируются через ⚙ Настройки. Требуют калибровки по реальным замерам на производстве.

| Параметр | По умолчанию | Значение |
|---|---|---|
| `t_straight_h_per_m` | 0.083 | ч/м прямого реза |
| `t_diagonal_h_per_cut` | 0.25 | ч на один косой рез (трапеция) |
| `t_arc_h_per_m` | 0.33 | ч/м дугового реза |
| `complexity_k` | 0.30 | коэффициент сложности малых деталей |
| `min_hours_per_part` | 0.15 | минимум часов на одну деталь |

## Steps Master

Открывается кнопкой **🪜 Ступеньки** на экране истории.

Использует общие таблицы матов, перфоуголка и кикплейта из основного конфига — никакого повторного импорта Excel. Дополнительно нужен файл Боковин (загружается один раз через главный диалог настройки файлов или кнопку в окне ступенек).

Расчёт: боковины × 2 + решётка (площадь × цена м²) + перфоуголок (€/м) + кикплейт (€/кг) + работа (€/час) + цинкование (+10% к весу).

Экспорт в Excel: `Ступени_YYYY-MM-DD.xlsx`.

## Анализ обрамления по кромкам

На реальных чертежах разные стороны детали могут иметь разный тип обрамления — например стандартный уголок сверху и тяжёлый кикплейт шириной 150 мм на трёх других сторонах. Программа передаёт это Claude как отдельную задачу анализа.

Claude изучает все выноски, подписи и размеры на кромках, определяет тип по контексту (не по конкретным словам — обозначения на чертежах бывают разные), считает длины и возвращает текстовое обоснование. Менеджер видит значок 🤖 на позиции, открывает редактирование и читает: *"Tun A на верхней стороне 690 мм и нижней правой части 403 мм = 1093 мм. Tun D (кикплейт 150 мм) на левой 400 + правой 400 + нижней левой 287 = 1087 мм."* — и либо подтверждает, либо правит вручную.
